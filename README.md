## Github Totals Count With Deno

Yes. I am learning deno. :)

### Usage

```bash
    deno task dev ibodev1 // Or your github username
```

#### With Compiled Exe File

```bash
    ./github ibodev1
```

or

```bash
    github.exe ibodev1
```
